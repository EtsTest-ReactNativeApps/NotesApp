<h1>Notes App</h1>
<p>Notes App is an android mobile application that allows the users to perform functions like add, search, delete, edit on your notes and save them on their mobile device as well. So, rather than jotting down points on a piece of paper, this application would indeed be a lot helpful to perform the required tasks. This application is developed using Facebook's open-source mobile application framework called React-Native.</p>

<h2>Installation Guide</h2>
<p>Install VSCode(Visual Studio Code) : https://code.visualstudio.com/download</p>
<p>Install NodeJS : https://nodejs.org/en/download</p>
<p>Install Expo Framework : https://expo.io/learn</p>
<h2>Execution of Application</h2>
<p>To run the application on Bundler: "expo start"</p>
<p>To run the application on Android Device/Emulator: "expo start --android"</p>
<h2>Deployment</h2>
<p>For building an apk file of the application : "expo build:android"</p>
<p>Link for downloading APK : https://github.com/MohitKambli/NotesApp/tree/master/apk</p>
<h2>Dependencies</h2>
<ul>
  <li>React Native Elements : https://react-native-elements.github.io/react-native-elements/</li>
  <li>Google Fonts : https://fonts.google.com</li>
  <li>Vector Icons : https://github.com/oblador/react-native-vector-icons</li>
</ul>
